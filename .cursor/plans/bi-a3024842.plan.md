# Add Direction Labels to Terrain Preview

## Changes

In [biome_edge_configurator.html](biome_edge_configurator.html), update `renderGridToCanvas()`:

1. Store `edgeDirection` on each cell in `applyEdgePatternsToGrid()`
2. After drawing the tile thumbnail, draw a small text label showing the direction abbreviation

**Label abbreviations:**

- "N", "S", "E", "W" for cardinal directions
- "NE", "NW", "SE", "SW" for corners
- "iNE", "iNW", "iSE", "iSW" for inner corners

The labels will be small white text with a dark background for readability, positioned at the bottom of each edge cell.

## Result

Each edge tile in the preview will show which direction pattern is being applied, making it easy to identify which button to click to change a specific tile.
