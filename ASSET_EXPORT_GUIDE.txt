ğŸ“¦ HOW TO EXPORT ANIMATION FRAMES

METHOD 1: SINGLE FRAME (F2)
1. Press F1 to enter DEBUG MODE
2. Position your character:
   - Use numpad 1-9 to lock direction (1=SW, 2=S, 3=SE, 4=W, 5=S, 6=E, 7=NW, 8=N, 9=NE)
   - Press F to enter FRAME MATCH MODE (for walk/run animations)
   - Use +/- to cycle through frames
3. Position the weapon using arrow keys
4. Press F2 to EXPORT the current frame as BMP
   - Auto-names as: running-halberd-north-frame0.bmp
   - Files save to your Downloads folder

METHOD 2: BATCH EXPORT ALL FRAMES (F3) - RECOMMENDED!
1. Press F1 to enter DEBUG MODE
2. Lock direction with numpad (e.g., 8 for NORTH)
3. Position weapon for frame 0
4. Press F3 to export ALL frames for this direction at once!
   - Exports running-halberd-north-frame0.bmp, frame1.bmp, frame2.bmp, frame3.bmp
   - All files go to Downloads
   - Weapon name is automatically included!

5. Then organize files into folders using organize_exports.bat
